Google Classroom MCP Setup

New to MCP? Learn what this tool does and how it connects to Claude. How It Works →

Step 1: Install the MCP Server

Run this command in your terminal:

npm install -g @salshah20/google-classroom-mcp

Step 2: Authorize with Google

Click the button below to connect your Google Classroom account:

Connect Google Classroom

Step 3: Add to Claude Desktop

After authorizing, you'll get a configuration snippet to paste into Claude Desktop.
