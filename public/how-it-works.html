How It Works

Connect Google Classroom to Claude using the Model Context Protocol (MCP).

What is MCP?

The Model Context Protocol (MCP) is an open standard that lets AI assistants like Claude securely connect to external data sources and tools. Instead of copying and pasting information, MCP servers give Claude direct, read-only access to your data -- so you can just ask questions in natural language.

What Can This MCP Server Do?

Once connected, Claude can access your Google Classroom data. Here's what you can ask it to do:

1
View Courses

List all your active courses, see course details, and check enrollment info.

2
View Rosters

See the students and teachers in each of your courses.

3
View Coursework and Submissions

List assignments, check due dates, and see student submissions and grades.

4
View Announcements

Read announcements posted to your courses.

5
Calculate Grades

See your grades across all courses and calculate overall percentages.

6
Track Missing and Upcoming Work

Find assignments that are past due or coming up soon across all your classes.

How the Connection Works

1
Install the MCP server

Run npm install -g @salshah20/google-classroom-mcp to install the server on your machine.

2
Authorize with Google

Sign in with your Google account and grant read-only access to your Classroom data. This generates a secure refresh token.

3
Configure Claude Desktop

Paste the generated configuration into your Claude Desktop config file. This tells Claude how to connect to the MCP server.

4
Start asking questions

Claude can now pull your Classroom data on demand. Just ask in natural language.

Example Prompts

Once set up, try asking Claude things like:

"List all my Google Classroom courses"
"Show me the students in my Biology class"
"What assignments are due this week?"
"What's my grade in History?"
"Which assignments am I missing?"
"Summarize recent announcements across all my courses"
